# Purnama

Memang tidak salah liat, judulnya adalah edisi weekend. Saya merangkap dua hari, sabtu dan minggu tanggal 6 dan 7. Saya lupa upload pas malam minggu karena terlalu asik bermain game _football manager_. Ya benar, saya terlalu asik bermain game, walaupun saya bukanlah seorang _gamer_ tapi saya punya adiksi dan atensi khusus pada game tersebut. Saya aktif bermain game tersebut sedari 2010. Entah apa formula yang ada pada game tersebut sehingga membuat saya memiliki adiksi yang kuat.

Disini saya mendapat pelajaran penting mengenai waktu dan manajemen waktu. Saya terlalu fokus pada hal sedang saya lakukan, padahal saya sedang memiliki suatu komitmen. Komitmen saya padahal cukup sederhana. Posting satu cerita satu hari. Saya pun lalai di hari sabtu, hari dimana saya libur dan tidak ada aktivitas padat.

Mungkin ini jadi tamparan nyata bagi diri saya, betapa saya sering lalai pada suatu hal jika saya sedang fokus dengan hal lain. Mumpung 2024 ini belum ada 1 purnama, saya akan pastikan hal-hal yang kurang baik seperti ini tidak terjadi hingga purnama ke-12 nanti pada akhir tahun.
